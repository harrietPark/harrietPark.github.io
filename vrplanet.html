Soular : The VR Planet Creator

C# | Unity Engine | VR | Game



Walkthrough Video


Soular: The VR Planet Creator was group coursework for the Virtual Reality module. Team members were Finn Tanner, Gun Lipe, Nicalia ThompSon, and myself. Everyone worked together for brainstorming the game concept and designing game mechanics. My main contribution was mostly programming and designing the user experience.

This app is a VR sandbox game that aims for self-expression and creativity. Users can create a unique planet using their hands. They can raise mountains, dig rivers, place objects, and do colouring, all with hand gestures.

Technology used :
-Hand Tracking,
-Hand Gesture Systems,
-Saving Features,
-Mesh Deformation,
-Procedural Planet Generation,
-Water Shader,
-Vertex Colouring


Why Soular : The VR Planet Creator?


Our initial idea was inspired by the Gravity Sketch, Tilt brush and the Lego set. In Gravity Sketch and Tilt brush, you have to create everything out from scratch. But our group wanted to make a game that does have a guideline the user can follow, but at the same time allows creativity. This game is similar to Lego sets but is more immersive in the sense that the user is inside the galaxy and creating a planet. Creating a planet in a space is something that you could not easily do in the real world. Also, this game doesn’t require controllers but uses hands for everything. So users can play this anywhere and anytime as long as they have the headset with them.
